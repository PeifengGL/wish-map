### Git Commit Message

`[<type>](trello #<number>): <Your Message>`

Example:
`[feature][#2]: Welcome Page/歡迎頁面`


### Git Branch

`<type>/<trello number>-<Your feature name>`

Example:
`feature/2-WelcomePage`

#### Type List

統一`小寫`

- feature （功能）
- fix （Bug）
- doc （寫文件）
- style （調整`格式`，不會影響到程式碼執行的變動，如：Tab、空格、換行）

#### Number

- Trello 上的卡片號碼
